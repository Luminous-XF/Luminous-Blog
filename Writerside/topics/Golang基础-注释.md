# 注释

在 Go 中提供了 C 风格的多行注释，以及 C++ 风格的单行注释。


## 单行注释
```Go
package main

func main() {
    // 单行注释
    // fmt.Println("Hello World")
}
```

## 多行注释
```Go
package main

func main() {
    /*
       fmt.Println("Hello World")
       fmt.Println("Hello World")
       fmt.Println("Hello World")
    */
}
```
